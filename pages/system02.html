技術內容詳述

一影像辨識與數據採集（OCR 模組）

採用 PaddleOCR 框架，整合至 Python Flask 後端服務。此模組負責對使用者掃描的商品圖片進行特徵文字擷取。

  • 即時效期辨識：準確抓取商品標籤上的「有效期限」數字與格式
  • 商品名稱辨識：透過關鍵字匹配輔助分類，自動將商品劃分至正確的大類別（例如肉類、蔬果類），為機器學習模型的特徵工程提供關鍵輸入

將核心數據採集的流程從人工輸入轉換為自動化掃描，大幅縮短消費者的操作時間，並確保後續 AI 模型的計算基礎數據是即時且精準的。
二機器學習與動態定價模型

本系統的核心價值在於利用多維度數據訓練的隨機森林迴歸模型，輸出具有科學依據的價格建議。

  • 集成優勢：採用 100 棵決策樹的集成策略，利用隨機抽樣機制，降低了單一模型過度擬合的風險，確保模型在不同銷售情境下的穩健性
  • 可重現性：訓練腳本中使用固定隨機數種子，保障模型每次訓練的結果一致性，有利於持續性維護與性能追蹤

模型成功整合了以下複合特徵，以模擬零售市場供需與環境影響：
  • 時效性特徵：由 OCR 辨識結果計算而得的剩餘保存期限（小時）
  • 環境特徵：當下溫度、人流量與天氣狀況
  • 營運特徵：貨架上庫存量與商品類別

透過 Flask API 接收數據後，模型即時推論出在當前情境下的「最佳建議折扣比例」，系統再以此計算並提供「建議合理價格」給予消費者參考。
